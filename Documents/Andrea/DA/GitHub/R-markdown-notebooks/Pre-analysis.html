Introduction

This is an exploratory analysis with the objective of checking for missing values, outliers or batch effects Regarding missing values, R’s default way of handling them is to drop them. However, the command na.action = na.warn informs about these instances. This exploratory analysis is performed on 17 plates. Each of these plates has 88 randomized plasma samples and 3 replicates of in-house plasma.

Data distribution and outliers

The exploratory analysis begins with the quality control using these in-house plasma replicates. Using both a histogram and a boxplot when initially exploring a dataset provides complementary insights into the distribution and characteristics of the data. In the first histogram, by looking at the normalized area of the peptide intensity peaks, we can assess if they follow a normal distribution.

#Missing values warning
options(na.action = na.warn)

#Read in data from Skyline's report
data <- read_csv2('Peptide Ratio Results.csv')

#Get rid of QTags
data <- subset(data, Protein != "QTag")

#Transform from character to numeric class
data$`Ratio To Standard` <- as.numeric(sub(",", ".", data$`Ratio To Standard`))
data$`Normalized Area` <- as.numeric(sub(",", ".", data$`Normalized Area`))
data$`Protein Abundance` <- as.numeric(sub(",", ".", data$`Protein Abundance`))
data$`DotProductLightToHeavy` <- as.numeric(sub(",", ".", data$`DotProductLightToHeavy`))

#Separation of plate (batch) and wells
data <- separate(data, `Replicate Name`, into = c("Plate", "Well"), sep = "_")
data$Plate <- as.numeric(sub(",", ".", data$`Plate`))

# Extract last three characters from 'Well'
data$Well <- substr(data$Well, nchar(data$Well) - 2, nchar(data$Well))
data_sub <- data[c("Plate", "Well", "Protein", "Peptide", "Normalized Area", "Ratio To Standard", "DotProductLightToHeavy")]

data_sub <- data_sub %>% 
  group_by(interaction(Plate, Well))

#Histogram of log10 of the normalized area
ggplot(data_sub, aes(x = log10(`Normalized Area`))) +
  geom_histogram(fill = 'purple', color = 'orange', bins = 100) +
  labs(title = "Normalized area distribution",
       x = "Log10 of normalized area",
       y = "Frequency") +
  theme_bw()

The histogram shows a skewed central tendency because of the many, very low values. It also indicates potential outliers towards these lower values. However, there do not seem to be gaps in the data. The boxplot, facilitates the identificaiton of the outliers and displays their relative distance from the bulk of the data, making it easier to compare distribution across the plates.

The same can be said about the boxplot, where the most notorious outliers, especially the ones tending to zero because of missing heavy isotopes have disappeared. What becomes aparent now is that there are some wells with much lower medians. Medians can be a robust measure for comparing samples due to their resistance to outliers -unlike means- and their ability to represent the central tendency of the data. Thus, in relatively large datasets like this one, such changes in the median between replicates can indicate bias that comes from technical variability.

Another way to look into technical variability is by using reference proteins like APOA1 and VTNC, whih are used to compare between plates because they are very stable, well-characterized and commonly measured in blood. Correlation between APOA1 and VTNC

To confirm this further, we can make a plot and see the correlation of their peptides’s median or the correlation of the peptide with the smallest inter-plate CV. As mentioned before, medians are robust to outliers, though they might lose some information about the dataset’s distribution compared.

# Calculate median 'Ratio To Standard' for each 'Peptide' within 'Protein'
apoa1_median <- apoa1_data %>%
  group_by(interaction(Plate, Well)) %>%
  summarize(Median_Ratio = median(`Ratio To Standard`))

vtnc_median <- vtnc_data %>%
  group_by(interaction(Plate, Well)) %>%
  summarize(Median_Ratio = median(`Ratio To Standard`))

#Merge both data frames into a single frame called correlation_data
correlation_data <- merge(apoa1_median, vtnc_median, by = "interaction(Plate, Well)")

# Perform linear regression using the new data frame to get residuals
lm_model <- lm(Median_Ratio.x ~ Median_Ratio.y, data = correlation_data)
residuals <- abs(residuals(lm_model))

# Find indices of samples with largest residuals
outliers <- order(residuals, decreasing = TRUE)[1:5] 

# Create a logical vector to indicate outliers
correlation_data$outlier <- row.names(correlation_data) %in% row.names(correlation_data[outliers, ])

# Plot correlation between median ratios with labels for top outliers
ggplot(correlation_data, aes(x = Median_Ratio.x, y = Median_Ratio.y)) +
  geom_point() +
  geom_smooth(method = "lm", se = FALSE,  formula = y ~ x) +
  geom_text(data = subset(correlation_data, outlier), aes(label = `interaction(Plate, Well)`), hjust = -0.2, vjust = -0.5) +
  labs(title = "Correlation of Median Ratios", x = "APOA1 Median Ratio", y = "VTNC Median Ratio") +
  theme_minimal()+
  stat_cor(label.x = 3, label.y = 0.85) +
  stat_regline_equation(label.x = 3, label.y = 0.80)

The correlation between both proteins is close to 1, confirming that they do follow the same pattern. This format also allows us to see plates that might have another type of problem that might not be aparent in the graphs before. It is also possible to use the peptide with the smallest CV instead of the median. When comparing the correlation between two proteins, choosing the smallest peptide coefficient of variation (CV) over the protein peptide median can offer specific advantages. Similarly to using the median, focusing on the peptide with the smallest CV, we are inherently reducing the influence of extreme variations or outliers present in other peptides. However, it is also providing insight into the behaviors or relationships that are more stable across conditions which can be later used for the actual samples and even future experiments.

# Calculate CV for each 'Peptide' within 'Protein'
apoa1_cv <- apoa1_data %>%
  group_by(Peptide) %>%
  summarize(Interplate_CV = sd(`Ratio To Standard`) / mean(`Ratio To Standard`) * 100)

vtnc_cv <- vtnc_data %>%
  group_by(Peptide) %>%
  summarize(Interplate_CV = sd(`Ratio To Standard`) / mean(`Ratio To Standard`) * 100)

# Find the smallest inter-plate CV for each protein
apoa1_min_cv <- apoa1_cv %>%
  filter(Interplate_CV == min(Interplate_CV)) %>%
  pull(Interplate_CV)

vtnc_min_cv <- vtnc_cv %>%
  filter(Interplate_CV == min(Interplate_CV)) %>%
  pull(Interplate_CV)

# Extract the peptide with the smallest CV
min_cv_peptides <- bind_rows(
  apoa1_cv %>% filter(Interplate_CV == apoa1_min_cv),
  vtnc_cv %>% filter(Interplate_CV == vtnc_min_cv))

# Calculate mean 'Ratio To Standard' for the smallest CV peptides
apoa1_smallest_cv <- apoa1_data %>%
  filter(Peptide %in% min_cv_peptides$Peptide) %>%
  group_by(interaction(Plate, Well)) %>%
  summarize(Ratio = mean(`Ratio To Standard`))

vtnc_smallest_cv <- vtnc_data %>%
  filter(Peptide %in% min_cv_peptides$Peptide) %>%
  group_by(interaction(Plate, Well)) %>%
  summarize(Ratio = mean(`Ratio To Standard`))

correlation_data <- merge(apoa1_smallest_cv, vtnc_smallest_cv, by = "interaction(Plate, Well)")

# Perform linear regression to get residuals
lm_model <- lm(Ratio.x ~ Ratio.y, data = correlation_data)
residuals <- abs(residuals(lm_model))

# Find indices of samples with largest residuals
outliers <- order(residuals, decreasing = TRUE)[1:5] 

# Create a logical vector to indicate outliers
correlation_data$outlier <- row.names(correlation_data) %in% row.names(correlation_data[outliers, ])

# Plot correlation between median ratios with labels for top outliers
ggplot(correlation_data, aes(x = Ratio.x, y = Ratio.y)) +
  geom_point() +
  geom_smooth(method = "lm", se = FALSE) +
  geom_text(data = subset(correlation_data, outlier), aes(label = `interaction(Plate, Well)`), hjust = -0.2, vjust = -0.5) +
  labs(title = "Correlation of ratios for peptide with smallest CV", x = "APOA1-VSFLSALEEYTK  ratio", y = "VTNC-DVWGIEGPIDAAFTR  ratio") +
  theme_minimal()+
  stat_cor(label.x = 3, label.y = 0.85) +
  stat_regline_equation(label.x = 3, label.y = 0.80)

The result is almost identical.

The conclusions that can be extracted from this preliminary exporation are: 1. The original data was skewed with some very extreme outliers, but this was mostly due to low quality peptides. 2. After filtering out dotp values under 0.85, the data showed a more centered normal distribution and the most extreme outliers disappeared. However, this also meant losing information. For instance, three peptides were measured in APOA1, but only two made the cut. This is not shown here, but found in the process. 3. Plates 3, 4 and 6 are subject to a similar batch effect that affects all three wells, while plates 31, 32 and 33 show two differing patterns. 4. APOA1 and VTNC’s behaviors are highly correlated, which also favors the decision of applying batch effect corrections as a following step. 5. Other plates that do not show such strong batch effects but that must be under strict observation are plate 7 and 22.
